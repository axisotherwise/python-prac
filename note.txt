python 가상환경 사용 이유
모듈간의 버전 문제로인해 새로운 모듈이 설치되지않거나 새로운 모듈이 설치되더라도 이전 코드가 동작하지 않을  수 있다.
가상환경은 아무런 모듈이 설치되어있지 않은 독립적인 파이썬 개발 환경이다.
모듈 설치 시 버전 문제를 최소화 할 수 있다.
가상환경을 만드는 방법은 여러가지가 있지만 python 명령어를 통해 만든다.

1. python -m venv "이름"
- 이름을 가진 독립적인 파이썬 가상환경을 만드는 명령어

- 이 명령어는 가상환경을 만드는 명령어일뿐 여기 만들어진 파일을 사용하라고 vscode에게 명령하지 않은 상태

- 가상환경 처음 생성 시 아무 모듈도 생성되어있지 않다.

- 가상환경 삭제는 가상환경 폴더 지우면 끝
